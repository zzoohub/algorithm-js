# 활성화 함수 & 손실 함수

## 활성화 함수 — 왜 필요한가?

활성화 함수가 없으면 신경망은 아무리 깊어도 **하나의 선형 변환**이다.

```
레이어 1: y₁ = W₁x
레이어 2: y₂ = W₂y₁ = W₂W₁x
레이어 3: y₃ = W₃y₂ = W₃W₂W₁x = Wx  ← 결국 하나의 행렬곱

활성화 함수를 넣으면:
레이어 1: y₁ = f(W₁x)
레이어 2: y₂ = f(W₂y₁)              ← 비선형! 복잡한 패턴 학습 가능
```

### ReLU — 현대 신경망의 기본

```
f(x) = max(0, x)

      ╱
     ╱
────╱──── x
   0

양수: 그대로 통과
음수: 0으로 차단

왜 좋은가:
- 계산이 빠르다 (비교 한 번)
- 기울기가 0 또는 1 → 기울기 소실 문제 완화
- 대부분의 CNN, Transformer에서 기본 활성화 함수
```

### Sigmoid — 확률 출력

```
f(x) = 1 / (1 + e^(-x))

    ─────── 1
   ╱
  ╱
 ╱
───────── 0

출력 범위: (0, 1) → 확률로 해석 가능
용도: 이진 분류의 마지막 층 ("고양이일 확률 87%")

단점: |x|가 크면 기울기 ≈ 0 (기울기 소실)
→ 은닉층에서는 ReLU를 쓰고, 출력층에서만 sigmoid
```

### Softmax — 다중 클래스 분류

```
입력: [2.0, 1.0, 0.1]  (raw scores, logits)
     ↓ softmax
출력: [0.659, 0.242, 0.099]  (확률 분포, 합=1)

"고양이 65.9%, 개 24.2%, 새 9.9%"

softmax(xᵢ) = e^xᵢ / Σ e^xⱼ

큰 값을 더 크게, 작은 값을 더 작게 → "승자 독식" 효과
```

### Tanh — RNN/LSTM의 선택

```
f(x) = (e^x - e^(-x)) / (e^x + e^(-x))

출력 범위: (-1, 1) — 중심이 0
→ Sigmoid보다 학습이 안정적 (출력 평균이 0에 가까움)
→ RNN, LSTM의 게이트에서 주로 사용
```

## 손실 함수 — "얼마나 틀렸는가?"

### MSE (Mean Squared Error) — 회귀용

```
L = (1/n) Σ (예측 - 정답)²

예측: [2.5, 0.5, 2.1]
정답: [3.0, 0.0, 2.0]
MSE = ((2.5-3)² + (0.5-0)² + (2.1-2)²) / 3 = 0.12

큰 오차에 더 큰 페널티 (제곱이니까)
→ 이상치에 민감
```

### Cross-Entropy — 분류용

```
L = -Σ yᵢ × log(ŷᵢ)

정답:  [0, 1, 0]  (원핫: 클래스 1이 정답)
예측:  [0.1, 0.8, 0.1]
L = -log(0.8) = 0.223  ← 정답 클래스의 확률만 본다!

예측이 [0.1, 0.99, 0.01]이면:
L = -log(0.99) = 0.01  ← 거의 확신 → 낮은 손실

예측이 [0.1, 0.01, 0.89]이면:
L = -log(0.01) = 4.6   ← 완전히 틀림 → 높은 손실
```

### Softmax + Cross-Entropy의 기울기

놀랍게도 결합하면 기울기가 극도로 간단해진다:

```
∂L/∂z = ŷ - y  (예측 - 정답)

예측: [0.659, 0.242, 0.099]
정답: [0, 1, 0]
기울기: [0.659, -0.758, 0.099]

"정답 클래스의 확률을 높이고, 나머지를 낮춰라"
```

이 간결함이 cross-entropy가 분류에서 표준인 이유 중 하나.

## 어떤 조합을 쓰는가?

| 문제 | 출력층 활성화 | 손실 함수 |
|------|-------------|-----------|
| 이진 분류 ("고양이/아님") | Sigmoid | Binary Cross-Entropy |
| 다중 분류 ("고양이/개/새") | **Softmax** | **Cross-Entropy** |
| 회귀 ("집값 예측") | 없음 (선형) | **MSE** |
| 은닉층 (대부분) | **ReLU** | — |

## 기울기 소실 문제 (Vanishing Gradient)

```
Sigmoid의 미분: σ'(x) = σ(x)(1-σ(x))  → 최대 0.25

10층 신경망에서 기울기가 역전파되면:
0.25 × 0.25 × ... × 0.25 = 0.25¹⁰ ≈ 0.000001

→ 앞쪽 레이어의 기울기가 거의 0 → 학습 안 됨!

ReLU의 미분: 1 또는 0 → 기울기가 축소되지 않음
→ 깊은 신경망이 가능해진 핵심 이유 중 하나
```
